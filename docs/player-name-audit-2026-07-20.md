# ATP/WTA 前 300 中文名审计

排名日期：2026-07-20

审计范围：ATP 单打前 300 与 WTA 单打前 300，共 600 人。逐条检查国内
媒体常用名、东亚球员原生汉字、复姓分隔、原语言发音及同名消歧。

## 结果

- 600/600 均有中文主显示名。
- 本轮修正 39 人；来源与链接保存在
  `data/player_name_overrides.json`。
- 17 个低曝光名字存在来源分歧或只有低等级中文资料，本轮不强制覆盖。
- 216 个名字仍属于受控音译或缺少稳定国内媒体用例，进入
  `data/player_name_review_queue.json`。该队列不阻断日报、知识帖或发布。

代表性修正：

- Learner Tien：蒂恩 -> 勒纳·钱（中国网球公开赛）
- Felix Auger-Aliassime：奥热-阿利亚西姆 -> 阿利亚西姆（央视网）
- Brandon Nakashima：纳卡什马 -> 中岛布兰登（新华社）
- Iga Swiatek：斯瓦泰克 -> 斯维亚特克（新华社）
- Ann Li：安·李 -> 李吉妮（国家体育总局）
- Joanna Garland：加兰 -> 葛蓝乔安娜（球员原生中文名）

## 规则

译名优先级：

1. 新华社、央视、国家体育总局及国内赛事官网。
2. 经人工核对的国内主流体育媒体。
3. 带 ATP、WTA 或 ITF 依据的中文球员资料。
4. 受控音译。

同一球员出现多套国内译名时，采用较新的高优先级来源。东亚球员优先核对
原生汉字及国内媒体实际姓名顺序。兄弟、父子或同姓球员保留足以消歧的名字
部分。没有稳定译名时保留暂定名并进入异步复核队列，不阻断内容时效。
